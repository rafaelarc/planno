Começando

1. Bem-vindo ao Planno!

O Planno é uma ferramenta completa para gerenciar suas tarefas e organizar sua vida. Este guia irá te ajudar a aproveitar ao máximo todas as funcionalidades.

2. O que o Planno pode fazer

O Planno oferece um conjunto completo de funcionalidades para organizar sua vida:

  • Gerenciar tarefas: Criar, editar, marcar como concluídas e excluir tarefas
  • Organizar por categorias: Agrupar tarefas por projetos ou áreas da vida
  • Sistema de tags: Marcar tarefas com até 3 etiquetas para melhor organização
  • Filtros combinados: Filtrar por categoria + tag simultaneamente (ex: "Trabalho + Urgente")
  • Filtros inteligentes: Encontrar tarefas por status, data, categoria ou tags
  • Busca rápida: Localizar tarefas por texto em qualquer campo
  • Ordenação dinâmica: Organizar tarefas por prioridade, data, status, categoria, tags, etc.
  • Configurações personalizáveis: Ajustar cores, notificações e exibição
  • Backup completo: Exportar e importar todos os dados e configurações
  • Guia integrado: Documentação completa dentro da aplicação
  • Armazenamento local: Seus dados ficam seguros no seu dispositivo

3. Interface Principal

  • Barra lateral: Filtros, categorias e tags
  • Lista de tarefas: Suas tarefas organizadas
  • Barra superior: Botões de ação e configurações

4. Primeiros Passos

  1. Configure seu nome nas configurações
  2. Crie suas primeiras categorias
  3. Adicione algumas tarefas
  4. Experimente os filtros e ordenação

Gerenciando Tarefas

Criando uma Tarefa

  1. Clique em "Nova Tarefa" no cabeçalho
  2. Preencha o título da tarefa
  3. Adicione uma descrição (opcional)
  4. Selecione a prioridade
  5. Escolha uma categoria
  6. Adicione até 3 tags
  7. Defina uma data de vencimento (opcional)
  8. Clique em "Salvar"

Editando uma Tarefa

  1. Clique no ícone de edição na tarefa
  2. Modifique os campos desejados
  3. Clique em "Salvar"

Marcando como Concluída

Clique na caixa de seleção ao lado do título da tarefa para marcá-la como concluída.

Excluindo uma Tarefa

Clique no ícone de lixeira na tarefa e confirme a exclusão.

Categorias

Criando Categorias

  1. Clique em "Gerenciar Categorias" na barra lateral
  2. Digite o nome da categoria
  3. Escolha uma cor
  4. Clique em "Adicionar"

Editando Categorias

Na janela de gerenciamento, clique no ícone de edição da categoria e modifique os dados.

Excluindo Categorias

Clique no ícone de lixeira. Categorias em uso não podem ser excluídas.

Filtrando por Categoria

Clique em uma categoria na barra lateral para ver apenas as tarefas dessa categoria.

Tags

Criando Tags

  1. Clique em "Gerenciar Tags" na barra lateral
  2. Digite o nome da tag
  3. Escolha uma cor
  4. Clique em "Adicionar"

Adicionando Tags às Tarefas

Ao criar ou editar uma tarefa, selecione até 3 tags na seção "Tags".

Filtrando por Tags

Clique em uma tag na barra lateral para ver apenas as tarefas com essa tag.

Gerenciando Tags

Use a janela de gerenciamento para editar ou excluir tags. Tags em uso não podem ser excluídas.

Filtros e Busca

Filtros por Status

  • Todas: Mostra todas as tarefas
  • Concluídas: Apenas tarefas finalizadas
  • Recorrentes: Tarefas que se repetem

Filtros por Data

  • Hoje: Tarefas para hoje
  • Em Breve: Tarefas da semana atual
  • Atrasadas: Tarefas atrasadas

Filtros Combinados

Você pode combinar categoria + tag simultaneamente!

  1. Clique em uma categoria (ex: "Trabalho")
  2. Clique em uma tag (ex: "Urgente")
  3. Resultado: Tarefas de "Trabalho" E "Urgente"

Exemplos de combinações:

  • "Trabalho + Urgente" → Tarefas de trabalho urgentes
  • "Pessoal + Importante" → Tarefas pessoais importantes
  • "Estudo + Revisão" → Tarefas de estudo para revisão

Indicadores Visuais

  • Título dinâmico: Mostra filtros ativos (ex: "Tarefas - Trabalho + Urgente")
  • Botão "Limpar Filtros": Aparece quando há filtros ativos
  • Destaque visual: Categorias e tags ativas ficam destacadas

Busca por Texto

Use a caixa de busca para encontrar tarefas por título, descrição ou tags.

Configurações

Informações Pessoais

Configure seu nome para personalizar a saudação no cabeçalho.

Personalização de Cores

Personalize as cores do aplicativo para combinar com seu estilo. Use o botão "Aplicar Cores" para testar.

Configurações de Exibição

Defina quantos dias de tarefas concluídas devem ser exibidos.

Ordenação Dinâmica

Configure como suas tarefas são organizadas na lista principal.

  • Por data: Data de criação ou vencimento
  • Por prioridade: Alta, média, baixa
  • Por status: Concluídas por último
  • Por categoria: Agrupadas por categoria
  • Por tags: Agrupadas por tags
  • Por título: Ordem alfabética

Escolha também a direção: crescente ou decrescente.

Salvando Configurações

Use o botão "Salvar Todas as Configurações" para aplicar todas as mudanças de uma vez.

Privacidade e Dados

Armazenamento Local

Todos os seus dados são armazenados localmente no seu navegador. Nada é enviado para servidores externos.

Backup dos Dados

RECOMENDADO: Use a função "Exportar Dados" nas configurações para fazer backup das suas tarefas, categorias e configurações.

Faça backup regularmente para não perder seus dados!

Restaurando Dados

Use "Importar Dados" para restaurar um backup anterior.

Segurança

Seus dados ficam seguros no seu dispositivo. Apenas você tem acesso a eles.

RESPONSABILIDADE DO USUÁRIO

IMPORTANTE: Você é 100% responsável pelos seus dados! Como o Planno armazena tudo localmente no seu navegador, apenas você tem controle sobre eles.

  • Se você limpar o cache do navegador, perderá todos os dados
  • Se trocar de navegador, perderá todos os dados
  • Se reinstalar o sistema, perderá todos os dados
  • Se o navegador corromper os dados, perderá tudo

CUIDADO: Limpeza de Dados

ATENÇÃO: A função "Limpar Todos os Dados" é IRREVERSÍVEL!

  • Você perderá TODAS as suas tarefas
  • Você perderá TODAS as suas categorias
  • Você perderá TODAS as suas tags
  • Você perderá TODAS as suas configurações
  • NÃO HÁ COMO RECUPERAR após clicar em "Limpar"

💡 DICA: Sempre faça backup antes de limpar os dados!
